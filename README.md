# [Simple Weather](https://matthewbeaud.in/simple-weather/)

Simple Weather is a web app I'm creating so I can avoid ad-infested websites when checking the weather.

## Features

- Current weather forecast
- 7 day forecast
- Hourly forecast (18 hours)

## What Simple Weather ISN'T

- A site cluttered with unnecessary content
- A site that runs slowly becuase of ads
- A site where you have to navigate multiple times to find information you want
